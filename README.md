# Network Packet Analyzer using Scapy

## 📌 Project Overview

This project is a Python-based Network Packet Analyzer developed using the Scapy library. It captures live network packets and displays important information such as Source IP, Destination IP, Protocol, Packet Length, and Timestamp.

---

## 🚀 Features

- Capture live network packets
- Display Source IP Address
- Display Destination IP Address
- Detect Protocol (TCP, UDP, ICMP)
- Show Packet Length
- Display Timestamp
- Simple and lightweight implementation

---

## 🛠️ Technologies Used

- Python
- Scapy

## ▶️ How to Run

### Install Scapy


```bash

```
